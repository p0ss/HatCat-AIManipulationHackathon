# Documentation

This folder contains research materials and submission documents for the hackathon.

## Structure

- **`references/`** - Source research papers and PDFs used as references
- **`submission/`** - Generated hackathon submission documents (DOCX, PDF)
- **`notes/`** - Working notes and drafts

## Usage

Drop PDF research papers into `references/` for context during document generation.
Final submission materials go in `submission/`.
